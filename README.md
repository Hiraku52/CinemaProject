# CinemaProject 🎬

A console-based cinema management system developed in Java as a long-term learning project focused on Object-Oriented Programming (OOP) and software design.

## Features

### Version 1 - Cinema Structure
- Create movies with their information.
- Assign movies to cinema rooms.
- Display all rooms and their assigned movies.
- Practice object relationships using composition.

### Version 2 - Interactive Menu
- Interactive console menu.
- User input using `Scanner`.
- Menu option validation.
- Program loop using `while`.

### Version 3 - Movie Search
- Search movies by name.
- Case-insensitive search (`equalsIgnoreCase()`).
- Display room number and movie information when found.
- Inform the user when a movie is not available.
